Negotiation Training in VR

A contract negotiation training in fully immersive VR with an AI powered virtual human as automated negotiation partner. Based on my master's thesis and project in collaboration with trainers.

React Chart.js Firebase Tailwind CSS

The Challenge

Real live negotiation training sessions are resource-intensive and require the presence of skilled trainers. The challenge was to create an automated system that allows users to practice negotiation skills in a virtual environment, providing realistic interactions without the need for a human partner.

The Solution

From an existing negotiation training concept, I developed a fully immersive VR application using Unity and Meta Quest 3. The virtual human negotiation partner is powered by OpenAI's GPT models, enabling dynamic and context-aware conversations. The system incorporates speech recognition and text-to-speech technologies to facilitate natural interactions, allowing users to engage in realistic negotiation scenarios anytime, anywhere.

The Process

Development of conversation flows in collaboration with negotiation trainers. First evaluation as remote controlled multiuser application. Implementation of AI-driven dialogue using GPT models. Integration of speech recognition and TTS for immersive VR experience.

The Evaluation

Evaluation and master thesis study.
