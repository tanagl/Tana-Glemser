Negotiation Training in VR

A contract negotiation training in fully immersive VR with a remote controlled virtual human as automated negotiation partner. Based on a real life negotiation training scenario in collaboration with trainers.[1]

React Chart.js Firebase Tailwind CSS

The Challenge

Real live negotiation training sessions are resource-intensive and require the presence of skilled trainers. The challenge was to create an immersive system that allows users to practice negotiation skills in a virtual environment, providing realistic interactions without the need for a human partner.

The Solution

From an existing negotiation training concept, I developed a fully immersive VR application using Unity and Meta Quest. The virtual human negotiation partner is remotely controlled by trainers, enabling dynamic and context-aware conversations. This setup allows users to engage in realistic negotiation scenarios anytime, anywhere, while expert trainers can guide the sessions efficiently.

The Process

Development of conversation flows in collaboration with negotiation trainers. First evaluation as remote controlled multiuser application.
