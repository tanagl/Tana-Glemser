Extended Minds - Soft Skill Training with Virtual Humans

A modular soft-skill training application in fully immersive VR with an AI powered virtual human as automated conversation partner.

OpenAI API Unity Meta Quest 3 Character Creator 4

The Project

Extended Minds was a planned start-up developed during the EXIST-Women program. The project aimed to create a modular soft-skill training application in fully immersive VR. The application combines generative AI with VR environments and humanoid AI agents to create a scalable, flexible, and cost-effective training platform. VR training for soft-skills has been shown to be effective and scalable [1]. In a team of three, we developed a prototype (based on my master thesis). As part of the EXIST-Women program we also developed a business plan and conducted market research, which showed promising potential for the product. The project was discontinued after the end of the EXIST program, but the prototype is still available for demonstration and further development.

The Solution

The application is capable of fully automating the simulation of a given scenario in VR. The environments were developed in Unity. Interaction takes place via speech recognition from the Meta Voice SDK. Answers were generated by GPT-models, using the OpenAI .NET client library from OkGoDoIt [2]. Test-to-speech was also implemented with the Meta Voice SDK. Information regarding the scenario is sent in an initial prompt. This workflow creates a natural flow of communication that provides learners with an authentic experience. Character Creator 4 models were used for the virtual humans [3].
