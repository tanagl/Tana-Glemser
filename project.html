Finance Dashboard App

A comprehensive tool for tracking personal expenses with real-time data visualization and CSV export features.

React Chart.js Firebase Tailwind CSS

The Challenge

Users often find it difficult to track monthly subscriptions. The goal of this project was to create a dashboard that automatically categorizes bank transactions and highlights recurring payments.

The Solution

I utilized the Plaid API to fetch transaction data securely. The frontend was built with React to ensure snappy transitions, while D3.js was used to render the complex spending graphs.
